Desenvolvimento Front-End

HTML CSS Javascript Apache

Html é um hipertexto de marcação, o qual serve para fazer a estrutura do site. Servindo mais para fazer demonstragem de conteúdo

CSS é uma folha de estilo para fazer alteração das marcações. Em resumo é para deixar a página mais estética.

Javascript é uma linguagem de programação utilizado para intereção da página com o usúario.

Apache é um servidor http, utilizado para serviço web.
